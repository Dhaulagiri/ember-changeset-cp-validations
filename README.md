## Requirements

- [ember-cp-validations](https://github.com/offirgolan/ember-cp-validations) v3.1.0 or above
- [ember-changeset](https://github.com/DockYard/ember-changeset)

## Installation

```
ember install ember-changeset-cp-validations
```
